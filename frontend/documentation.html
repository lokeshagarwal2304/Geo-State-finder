Introduction

Welcome to the CountryFinder AI API documentation. Our API allows developers to programmatically validate phone numbers, identify carriers, detect line types (Mobile, Landline, VoIP), and enrich user profiles with identity intelligence across 230+ countries.

Authentication

Authenticate your API requests by including your API key in the header of every request.

Terminal 
curl -X GET "https://api.countryfinder.ai/v1/validate/+14155550123" \
  -H "Authorization: Bearer YOUR_API_KEY"

API Endpoints

POST /predict

Validate a phone number and retrieve basic carrier details.

Request Body

{
  "input": "+919876543210",
  "deep_search": false
}

Python Example 

import requests

url = "http://localhost:8000/predict"
payload = {
    "input": "+14155550123",
    "deep_search": True
}
headers = {"Content-Type": "application/json"}

response = requests.post(url, json=payload, headers=headers)
print(response.json())

Response Structure

The API returns a JSON object containing validation status and enrichment data.

{
  "success": true,
  "formatted": "+1 415-555-0123",
  "valid": true,
  "country": "United States",
  "code": "+1",
  "carrier": "AT&T Mobility",
  "type": "Mobile",
  "name": "Given Name (Deep Search Only)",
  "risk_score": 0.05,
  "confidence": 0.99
}

Error Codes

Code Description
400 Bad Request Missing or invalid phone number format.
401 Unauthorized Invalid or missing API key.
429 Too Many Requests Rate limit exceeded for your plan.
500 Internal Server Error Something went wrong on our end.
